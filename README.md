## Talisman programming language

This is an example of the talisman programming language:

```hello.t
set x = 10

if x == 10 then
    print("it is a 10")
else if x == 3 then
    print("it is a 3")
end

// comments
// declaring methods
fn add(x, y)

end

// for loops
for set i=0; i<x; i++ do
    print(i)
end

// while loops
set a = 0
for x < 20 do
    a = a + 1
    x = x + 1 
end
```
